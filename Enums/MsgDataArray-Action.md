These enumeration type values are used to control what type of system the items should be used in via [MsgDataArray](Packets/MsgDataArray).

```cs
ComposeItem = 0,
Accessory = 1,
ComposeSteedOriginal = 2,
ComposeSteedNew = 3,
QuickCompose = 4,
QuickComposeMount = 5,
UpgradeItemLevel = 6,
UpgradeItemQuality = 7
```
